## 功能

- [x] 扫码添加cookie到Bilibili.sqlite3数据库
- [x] 添加cookie后刷新脚本的env配置文件
- [x] pushplus群组推送
- [x] 多次扫码cookie更新替换
- [x] 匹配DedeUserID删除库内cookie
- [x] 增加手动录入ck页面，方便自己浏览器抓ck录入
- [ ] 删除同时刷新脚本配置文件删除
- [ ] 定时识别库内cookie是否有效
- [ ] 一对一通知推送
- [ ] 个人自定义配置
- [ ] 账号管理面板
